
# Vocabulary Tester

Small terminal/console app created to help student with learning vocabulary for tests.


## Install

```console
git clone https://github.com/gastyCode/vocabulary-tester.git
cd vocabulary-tester
pip install -r requirements.txt
python .\main.py
```
